# TinyFish AI Hiring Dashboard

AI-powered hiring assistant that analyzes GitHub, resume, and skills to help recruiters evaluate candidates faster.

## Tech Stack
- FastAPI backend (Python)
- TinyFish Agent API
- React frontend
- Tailwind CSS
- Recharts + Framer Motion

## Features
- Resume upload & analysis
- GitHub activity analysis
- Skill verification
- Candidate scoring dashboard
- AI interview question generation

## How to Run

### Backend
cd TinyFish
venv\Scripts\activate
uvicorn main:app --reload

### Frontend
cd hiring-dashboard
npm start
